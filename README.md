## Flask 速通案例

系统地学习并快速掌握 Flask 框架，从快速启动到高级功能的实现，涵盖了 Flask 开发的各个方面。通过一系列有条理的示例和练习，帮助开发者深入理解 Flask 的核心概念和最佳实践。

### 依赖解决

```bash
pip install poetry  # 国内下载缓慢请自行使用镜像源下载

# 接下来安装环境所依赖的包
poetry lock
poetry install
```
